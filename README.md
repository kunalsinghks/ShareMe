## üìñ How it Works

ShareME creates a temporary, encrypted "bridge" directly to your local computer using **Cloudflare Argo Tunnels**.

1. **Local Backend**: High-performance **FastAPI** indexes your shared workspace.
2. **The Bridge**: Secured tunnel from your machine to a unique `trycloudflare.com` URL.
3. **No Config**: Bypasses **CGNAT** and firewalls without manual router configuration.
4. **Security**: Mandatory E2EE transit and optional Vault password protection.
